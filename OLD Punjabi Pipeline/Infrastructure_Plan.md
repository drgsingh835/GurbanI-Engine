# Punjabi Guftar: A-Z Channel Infrastructure Plan

This document outlines the complete organizational and technical framework required to run a high-fidelity, academically-focused YouTube channel.

## 📂 Phase 1: Folderization (The "Digital Production Plant")
We will use a hierarchical structure on `F:\Punjabi_Guftar_Workspace` to ensure no data is lost and automation is easy to implement.

### 01_Scholarly_Foundation
*   **01_Etymology_Archives**: Digital dictionaries, linguistic papers (PDFs).
*   **02_Reference_Manuscripts**: Images/scans of old Punjabi literature.
*   **03_Curriculum**: Structural plans for educational series.

### 02_Scripting_Bay
*   **01_Active_Drafts**: Current working markdown files.
*   **02_Master_Archive**: Final, approved scripts in Gurmukhi.
*   **03_LLM_Prompts**: Specialized prompts for research and scripting automation.

### 03_Visual_Laboratory
*   **01_Thumbnails**: Design project files (PSD/AFFINITY) and final JPEGs.
*   **02_Calligraphy_Assets**: Transparent PNG/SVG files of Punjabi words.
*   **03_Background_Media**: AI-generated B-roll, stock footage, and textures.
*   **04_Typography_Cards**: Pre-rendered overlays for video text.

### 04_Audio_Recording_Studio
*   **01_Raw_VO**: Unedited voiceover recordings.
*   **02_Clean_VO**: Noise-reduced and equalized master audio.
*   **03_Soundscapes**: Background music (Lo-fi/Traditional) and sound effects (SFX).

### 05_Video_Production_Plant
*   **01_Project_Files**: Premiere Pro, DaVinci, or CapCut project files.
*   **02_Automation_Scripts**: FFmpeg `.bat` or Python scripts for batch processing.
*   **03_Rough_Cuts**: Iterative exports for review.
*   **04_Final_Master_Exports**: High-bitrate files ready for upload.

### 06_Distribution_and_Growth
*   **01_Metadata_Vault**: CSV/Markdown files containing SEO Titles, Tags, and Descriptions.
*   **02_Community_Content**: Polls, short-form scripts, and teaser images.
*   **03_Performance_Logs**: Monthly analytics tracking.

### 07_Governance
*   **01_Style_Guides**: The "Academic Punjabi" constitution.
*   **02_Asset_Inventory**: Links and licenses for all fonts and music.

## 🚀 Phase 3: The "A-Z" Workflow
1.  **Define Concept**: Choose a word/theme (e.g., *Guftar*).
2.  **Automated Research**: Agent generates etymology and script draft in `01_Scholarly_Foundation`.
3.  **Human Audit**: You refine the script and record audio in `04_Audio_Recording_Studio`.
4.  **AI Asset Generation**: Agent generates 5 thumbnail concepts and 3 B-roll clips in `03_Visual_Laboratory`.
5.  **Automated Edit**: FFmpeg script strips silence and overlays the "Gurmukhi Word Card" automatically.
6.  **Final Polish**: You do a 5-minute final check in the Video Editor.
7.  **SEO Deployment**: Agent generates the "Metadata Vault" entry for easy copy-pasting.
8.  **Upload**: Publish to YouTube Studio.
